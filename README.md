# Brazilian E-Commerce Sales Analysis Using SQL

## Project Overview

This project analyzes the Brazilian E-Commerce Public Dataset (Olist) using MySQL. The objective is to extract meaningful business insights related to customer behavior, sales performance, payment methods, product categories, and regional trends.

The project demonstrates practical SQL skills including data exploration, table joins, aggregations, grouping, ranking, and business KPI analysis.

---

## Tools Used

* MySQL
* MySQL Workbench
* Power BI
* GitHub

---

## Dataset

Dataset Source: Brazilian E-Commerce Public Dataset by Olist

Tables Used:

* olist_customers_dataset
* olist_orders_dataset
* olist_order_items_dataset
* olist_order_payments_dataset
* olist_products_dataset

---

## Database Schema

Customers → Orders → Order Items → Products

Customers → Orders → Payments

Key Relationships:

* customer_id links Customers and Orders
* order_id links Orders, Payments, and Order Items
* product_id links Order Items and Products

1. What is the total revenue generated?
